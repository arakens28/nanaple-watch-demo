# 事業計画書 — 月商10億円プロジェクト

> AI上司 3名（ChatGPT / Gemini / Claude）＋ 人間スタッフ 3名（マーケ1・営業2）で、
> ホリエモンAI学校の「法人受講」「FC加盟」を軸に **月商10億円** を目指す。

最終更新: 2026-06-09 ／ ステータス: ドラフト（数値は検証前のモデル仮説）

---

## 0. このドキュメントの読み方（先に正直な話）

月商10億円（年商120億円）は、6人規模のチームにとって **極めて野心的な目標** です。
本計画書は「気合で行ける」とは書きません。代わりに、

1. **10億円に到達するには各レバーがいくつ必要か**を逆算で数式化し、
2. その数字が現実的かどうかを段階的に検証できる **マイルストーン** に落とし込み、
3. 6人＋AIで回すための **役割分担とツール構成** を定義します。

「まず月商1,000万円 → 1億円 → 10億円」と桁を上げていく前提で、各フェーズのゲート条件を置いています。

## 1. 組織体制

### 1.1 人間スタッフ（3名）

| 役割 | 人数 | 主担当 | 主要ツール |
|------|------|--------|-----------|
| マーケター | 1 | 集客・LP・広告・SNS運用・ファネル全体設計 | X / Lステップ / 広告管理画面 |
| セールスA | 1 | インバウンド対応・個人/法人クロージング | HubSpot / Crisp / オンライン商談 |
| セールスB | 1 | FC加盟・法人アウトバウンド・大型案件 | HubSpot / メール / 商談 |

人間は「最終判断・対人折衝・例外処理・倫理/法務チェック」に集中。
反復作業（コピー量産・スコアリング・一次対応・レポート生成）はAIに委譲する。

### 1.2 AI上司（3名）

3つのLLMを **役割で使い分ける**。単なる横並びではなく、得意領域で分業させる。

| AI上司 | 担当領域 | 使いどころ |
|--------|----------|-----------|
| **ChatGPT** | クリエイティブ・コピー・台本 | 広告コピー、X投稿、セールス台本、LP文面の大量生成とABテスト案 |
| **Gemini** | リサーチ・データ分析・市場調査 | 競合/市場リサーチ、長文資料の要約、KPIデータの分析、検索連動の最新情報収集 |
| **Claude** | 統括・意思決定・品質管理 | 全体戦略の整合性チェック、各案のレビュー、最終アクションプラン決定、長文ドキュメント整備 |

> 設計思想: ChatGPT＝発散（量）、Gemini＝検証（事実とデータ）、Claude＝収束（判断と品質）。
> 3者の出力を突き合わせることで、単一モデルのハルシネーションや偏りを相互補正する。

### 1.3 意思決定フロー

```
人間（タスク投入）
   ↓
ChatGPT（アイデア・コピー大量生成）  ┐
Gemini（リサーチ・データ裏付け）     ├→ Claude（統合・レビュー・最終決断）
過去KPI / ツールのデータ             ┘        ↓
                                     人間が承認 → 実行（ツールで自動化）
                                              ↓
                                     結果データを再び投入（ループ）
```

## 2. プロダクトと収益源

ホリエモンAI学校のアセットを軸に、単価と継続性の異なる4階層を持つ。

| 階層 | プロダクト | 想定単価 | 性質 |
|------|-----------|----------|------|
| エントリー | 個人オンライン受講 | 月 1〜3万円 | 月額・大量・継続課金 |
| ミドル | 法人受講（研修パッケージ） | 1社 月20〜100万円 | 法人・複数席・安定 |
| ハイ | **FC加盟**（スクール開校権） | 加盟金 数百万円＋月額ロイヤリティ | 高単価・スケールの主役 |
| 付帯 | イベント/コンサル/物販 | 変動 | LTV底上げ |

**スケールの主役は FC加盟と法人受講**。個人受講は「リスト獲得と信頼構築の入口」と位置づける。

## 3. 月商10億円の逆算モデル（仮説）

> ⚠️ 以下は **検証用のモデル**。実数は走りながら更新する。複数シナリオを併記する。

### シナリオA：FC主導型

| 項目 | 数値 |
|------|------|
| FC加盟1件あたり月次貢献（ロイヤリティ＋初期償却按分） | 100万円 |
| 必要な稼働FC数 | **約700件** |
| 法人受講（平均30万円/月）の社数 | 1,000社（=3億円） |
| 個人受講（平均2万円/月） | 30,000人（=6億円） |

→ FC 700件は数年がかりの規模。**単独で10億は非現実的**、複合が前提。

### シナリオB：複合型（推奨）

| 収益源 | 単価/月 | 件数 | 月商 |
|--------|---------|------|------|
| 個人受講 | 2万円 | 25,000人 | 5.0億円 |
| 法人受講 | 40万円 | 750社 | 3.0億円 |
| FC加盟ロイヤリティ | 80万円 | 150件 | 1.2億円 |
| FC新規加盟金（償却） | — | 月10件×初期500万 | 0.5億円 |
| イベント/付帯 | — | — | 0.3億円 |
| **合計** | | | **10.0億円** |

### 現実度チェック（重要）

- 個人受講 **25,000人** の継続課金は、解約率を月5%に抑えても毎月1,250人の純増が必要。
  → これは集客の主戦場。X＋広告＋Lステップの自動ナーチャで CPA を成立させられるかが生命線。
- 営業2名で法人750社・FC150件を **直接さばくのは不可能**。
  → 「AI一次対応 → 高確度のみ人間クローズ」「FCオーナー自身が現場を回す多層構造」が必須。

**結論**: 10億円は「6人が頑張る」では届かない。**FCと受講生をレバレッジにした"組織の自己増殖"** が成立してはじめて射程に入る。本計画はそこへ至る階段を定義する。

## 4. 集客・販売ファネル

```
認知        X（毎日投稿・バズ設計）/ ホリエモン関連メディア / 広告
  ↓
リード獲得   無料コンテンツ・ウェビナー → Lステップ友だち登録
  ↓
ナーチャ     Lステップでステップ配信（教育・事例・実績）
  ↓
一次対応     Crisp（サイトチャット）＋AIが24h自動応答・FAQ・予約
  ↓
商談         HubSpotでスコアリング → 高確度を営業へアサイン
  ↓
クロージング  個人=自動決済 / 法人・FC=人間営業
  ↓
継続・拡大    HubSpotで顧客管理 → アップセル（個人→法人→FC）
```

各段でAIが下書き・スコアリング・一次対応を担い、人間は判断と対人折衝に集中する。

## 6. フェーズ別ロードマップ（ゲート方式）

各フェーズは **次に進む条件（ゲート）** を満たしてから移行する。未達なら同フェーズで改善を回す。

### Phase 0（0〜1ヶ月）: 基盤構築
- [ ] AI上司3名のワークフロー確立（本リポジトリのデモUIで運用テスト）
- [ ] X / Lステップ / HubSpot / Crisp の最小構成を接続
- [ ] エントリー商材のLP・決済・ステップ配信を1本完成
- **ゲート**: 自動で「認知→登録→決済」が1件でも回る

### Phase 1（1〜3ヶ月）: 月商1,000万円
- [ ] 個人受講の継続課金を黒字CPAで獲得
- [ ] Crisp＋AIの一次対応で営業の工数を削減
- **ゲート**: 月商1,000万円・解約率10%以下・CPA<LTVの3分の1

### Phase 2（3〜9ヶ月）: 月商1億円
- [ ] 法人受講パッケージを商品化し10社以上
- [ ] FC加盟スキームを設計し最初の数件を獲得
- **ゲート**: 月商1億円・法人/FCの再現性のあるクローズ手順が確立

### Phase 3（9〜24ヶ月）: 月商10億円へ
- [ ] FCオーナーが自走する多層組織を構築（テンプレ・研修・支援AIを提供）
- [ ] 個人2.5万人規模の集客エンジンを安定運転
- **ゲート**: シナリオB（§3）の各レバーが個別に成立し、合算で10億円

## 8. リスクと前提

- **目標規模の現実性**: §3のとおり10億は複合かつ多層化が前提。マイルストーン未達なら目標時期を見直す。
- **属人化**: 営業2名がボトルネックになりやすい。AI一次対応とFC自走で分散する。
- **法務・景表法・特商法**: 教育/FC商材は表記規制が厳しい。誇大表現を避け、人間が最終チェック。
- **AI出力の品質**: ハルシネーション対策として3モデル相互チェック＋人間承認を必須とする。
- **プラットフォーム依存**: X・LINE・各ツールの規約変更リスク。チャネルを分散する。
