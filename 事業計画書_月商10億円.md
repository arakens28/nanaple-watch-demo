## 0. このドキュメントの読み方（先に正直な話）

月商10億円（年商120億円）は、6人規模のチームにとって **極めて野心的な目標** です。
ただし現在地は月商4,000万円であり、**ゼロイチではなく「25倍のスケール」**が課題です。
本計画書は「気合で行ける」とは書きません。代わりに、

1. **4,000万円 → 10億円に到達するには各レバーがいくつ必要か**を逆算で数式化し、
2. その数字が現実的かどうかを段階的に検証できる **マイルストーン** に落とし込み、
3. 6人＋AIで回すための **役割分担とツール構成** を定義します。

「現状4,000万円 → 1億円 → 3億円 → 10億円」と段階的に桁を上げる前提で、各フェーズのゲート条件を置いています。

## 2. プロダクトと収益源（公式資料の実データ）

ホリエモンAI学校（堀江貴文 共同創業・取締役）のアセット。AI未経験者向け（受講者の95%が非エンジニア）の伴走型オンラインスクール。240+の動画講義・チャット講師サポート・毎月の伴走MTG・ChatGPT資格（文章生成AI能力検定）。

| 階層 | プロダクト | 実際の価格 | 性質 |
|------|-----------|-----------|------|
| 個人受講 | 個人オンライン受講 | — | 入口（法人プランは1名不可） |
| **法人受講** | 法人プラン | **¥31万/年/人**（税込¥34.1万）・**4名以上推奨** | 年額・複数席 |
| **FC加盟** | 業界別オンライン学部／地域別オフライン校 | 初期 **受講料¥1,200万＋加盟金¥200万**、以降 **売上の50%**を受領 | 高単価・主エンジン |

### 2.1 重要：助成金が最大の販売武器

- 法人受講は **厚労省「人材開発支援助成金」で受講料の最大75%（大企業60%）が助成**。
  例）4名受講 ¥124万 − 助成金 ¥96万 = **実質¥28万**（最安1人 ¥5,833/月〜）。
- ただし申請は素人には困難 → **社労士（TRIPORT）面談が必須ステップ**。この"実質負担が激減する"訴求が成約率を支えている。
- 典型的な法人案件 ≒ **5名 = 年間¥155万**（§3.4の「1件¥150万」の正体）。年額契約のため **更新で継続化の余地**がある。

### 2.2 FCの位置づけ

- 加盟条件: 「特定エリアの販路」「特定業界の販路」「社員42名以上の法人」のいずれか1つ以上。
- **FCは受講企業のみがなれる**（まず¥1,200万の受講 → 3ヶ月後にFC契約）。
- **FCが新たなFCを勧誘してもOK** → 組織が自己増殖する設計。
- 2025年7月時点12社 → 現在18社。**エリア別・業界別とも100校まで募集中**。

**スケールの主役は FC**。法人受講はFCが現場で売る商品でもある（売上を会社とFCで50:50）。個人受講は入口。

## 3. 月商10億円の逆算モデル（現状の内訳ベース）

### 3.0 現状の売上構成（損益帳票 2026年5月）

| 収益源（チャネル） | 月商 | 比率 | 性質 |
|--------|------|------|------|
| 法人・本部直販 | ¥2,458万 | 55% | B2B・営業2名が直接獲得 |
| 法人・FC経由 | ¥1,312万 | 30% | FCが現場で販売（売上を会社とFCで折半） |
| 個人・本部/FC | ¥427万 | 10% | 入口・小口 |
| その他 | ¥244万 | 5.5% | 企業研修・経営塾など |

> **重要な事実**: この事業は **法人で85%** のB2B型。販売チャネルが「本部直販」と「FC経由」の2系統。
> 「個人を大量集客する消費者ビジネス」ではない。勝ち筋は **①本部の法人営業を磨く ②FC経由の販売網を増やす**の2軸。個人受講は売上の柱ではなく入口。

### 3.1 目標構成（推奨）：2チャネルを伸ばす

| 収益源（チャネル） | 現状/月 | 目標/月 | 倍率 | 役割 |
|--------|---------|---------|------|------|
| 法人・FC経由 | ¥1,312万 | **¥5.0億** | 約38倍 | **主エンジン**（FC網が自己増殖） |
| 法人・本部直販 | ¥2,458万 | **¥4.0億** | 約16倍 | 営業＋AIで歩留まり改善 |
| 個人・その他 | ¥671万 | **¥1.0億** | 約15倍 | 入口・付帯 |
| **合計** | **¥4,441万** | **¥10.0億** | **約23倍** | |

### 3.2 法人受講の単価（台帳154契約の実績）

- 平均単価 **¥137万/契約**（税込・152件記入）、平均 **9.1名/契約**（受講者計1,398名）。
- 契約ペースは加速: 2023年58件 → 2024年60件 → 2025年は月14件規模へ。
- **助成金が成約の鍵**: 記入済みのうち TRIPORT等で申請68件 vs 申請なし14件（**8割超が助成金活用**）。
- 地域は **東京34%** 偏在（大阪9%・兵庫/京都/神奈川/福岡 各5%）。→ 地方は伸びしろ＝地域別FCの出番。

### 3.3 最重要：年額契約の「ランオフ」を止めることが先決

損益帳票の先行きは、現状ペースだと **月商が¥4,441万→¥2,063万（8ヶ月で約半減）**。理由は **法人受講が年額契約で、満了すると剥落** するから。

- **本部直販（法人）= フローに近い**: 毎月新規を売り続けないと減る。①面談化率改善（§3.4）②**更新率を上げる**（年額の継続化）が生命線。
- **FC経由 = ストック化しやすい**: FCが現場で売り続ける＆増えるので、本部の営業工数を増やさず売上が積み上がる。**だからFCが主エンジン**。
- **個人**: 柱にしない。法人・FCへの入口として維持。

**結論**: 「売って終わり」を「売って積み上がる」に変える。具体的には **(A) FC網の拡大（自己増殖）** と **(B) 法人の更新率向上＋面談化率改善**。物量だけで25倍は不可能。

### 3.4 法人ファネルの実数分析（6ヶ月: 2025/12〜2026/5）

```
インバウンド  リード576件 ──31.2%──▶ 面談すみ180件 ──36.1%──▶ 契約65件
（広告460/オーガニック116）          └─ 面談できず 396件（リードの約68%！）  全体歩留 11.2%

アウトバウンド 面談346件（外注アポ @¥3万）──4.0%──▶ 契約14件   CAC ≒ ¥74万/件

6ヶ月 合計79契約 ≒ 月13件 × ¥150万 ≒ ¥2,000万/月  ✓
リードタイム: 1ヶ月以内30% / 2〜3ヶ月60%（ほぼ3ヶ月で決着）
```

**ボトルネックと打ち手（優先度順）**

1. 🔴 **最大の穴＝面談化率31.2%**（リード576のうち396件が面談に至らず）。
   → **Crisp＋Lステップ＋AIで即レス・自動日程調整**し面談化率を50%へ。
   試算: 31%→50%で面談180→288件、契約65→104件（**+39件／法人月商 約¥2,000万→¥2,950万**）。**追加広告費ゼロで＋50%**。
2. 🟠 **インバウンドはアウトバウンドの9倍効率**（面談→契約 36.1% vs 4.0%）。
   → 予算と工数をインバウンドに寄せる。外注アポ（@¥3万・CAC¥74万）はAIで事前選別し、確度の低い架電を削減。
3. 🟡 **広告チャネル最適化**（広告460件: meta/google/Yahoo中心、X 2件）。
   → CPA×契約率でチャネル別ROIを可視化し、勝ち筋に再配分。X由来が極小なのは伸びしろ。
4. 🟢 **パイプライン管理**（面談すみ見込み10社=¥1,500万・確率50%）。
   → HubSpotで案件ごとに確度×金額を可視化し、3ヶ月の決着サイクルに合わせて追客を自動化。

> ⚠️ 要確認: 広告の総費用（インバウンドCAC算出に必要）、法人受講の更新率、FCの月次純増ペース。

### 3.5 FCフライホイールの経済性と「構造的な天井」

**FC1校の経済性（資料ベース）**
- 会社の初期収入: 受講料¥1,200万＋加盟金¥200万 = **¥1,400万/校**（FC加盟時）
- 会社の継続収入: FCが売る研修・コンサルの **売上の約50%**。FC1校の年間売上目標¥5,000万 → **本部取り分 ≒ ¥2,500万/年/校**（成熟時）
- 現状: FC経由¥1,312万/月 ÷ 18社 ≒ **本部取り分¥73万/月/校（≒¥875万/年）** → 成熟目標¥2,500万の **約35%**。**成熟するだけで約3倍の伸びしろ**。
- **FCがFCを勧誘可** → 販売網が自己増殖。台帳でも28〜35契約がFC関連。

**⚠️ 構造的な天井（正直な試算）**

| レバー | 上限の目安 | 月商換算 |
|--------|-----------|----------|
| FC経由（100校 × 成熟¥2,500万/年） | 募集上限100校 | **≒¥2.1億/月** |
| 本部直販 法人（営業＋AIで歩留まり改善） | — | ≒¥0.5〜1億/月 |
| FC加盟金（新規加盟ペース次第） | 年数十校 | 数千万/月 |
| 個人・その他 | — | ≒¥0.1億/月 |

→ **現行モデルの延長線上の天井は、ざっくり月商¥3〜4億**。**月商10億円は現行ルールのままでは構造的に届かない**。
10億円を本気で狙うなら、次のいずれか（複数）が必要：

1. **100校の募集上限を引き上げる**（または海外・新業界へ拡張）
2. **FC1校あたりの生産性を¥5,000万/年より大きく引き上げる**（AI支援で受注力を底上げ）
3. **新しい収益ライン**（経営塾の拡大、AIプロダクト/SaaS化、AIクチコミ等の付帯を主力化）

> つまり「いまの勝ち筋を磨く」のはPhase 2〜3（月商¥3〜4億）まで。**10億円はモデル自体の拡張**が前提。ここを経営に握ってもらう必要がある（§9）。

## 4. 集客・販売ファネル（法人・FC主導）

主役は **法人受注とFC加盟**。個人受講は「入口」であり、最終的に法人・FCへ引き上げる。

```
認知        X（実績・ノウハウ発信）/ ホリエモン関連メディア / 法人向け広告
  ↓
リード獲得   無料コンテンツ・ウェビナー・個人受講 → Lステップ／HubSpotに蓄積
  ↓
ナーチャ     Lステップ配信（事例・成果・FC説明会・法人導入事例）
  ↓
一次対応     Crisp（サイトチャット）＋AIが24h自動応答・FAQ・説明会予約
  ↓
スコアリング  HubSpotで「法人見込み」「FC見込み」を判定 → 高確度を営業へ
  ↓
クロージング  法人=セールスA / FC=セールスB（高単価ほど人間が対応）
  ↓
拡大ループ   法人→追加席・更新 ／ FCオーナー→次のFCを紹介（自己増殖）
```

各段でAIが下書き・スコアリング・一次対応を担い、**人間2名の営業は法人・FCの高確度クロージングに集中**する。
「個人→法人→FC」のアップセル設計が、5%の個人受講を10億円エンジンの燃料に変える。

## 8. リスクと前提

- **目標規模の現実性**: §3のとおり10億は複合かつ多層化が前提。マイルストーン未達なら目標時期を見直す。
- 🔴 **営業赤字（最優先）**: 法人エンティティはピーク月でも営業赤字（年間 -¥6,070万見込み）。販売手数料/広告/外注が重い。**規模より先に各月の黒字化**（§3.6）。
- 🔴 **ランオフ（年額契約の剥落）**: 前受収益¥278Mの取り崩しで、新規・更新が止まると売上が急減。**新規＋更新率の確保が生命線**。
- ⚠️ **債務超過は会計上の見え方**: 主因は前受収益（前払い）。資金繰り危機ではないが、新規が細ると実態も悪化する（§3.6）。
- **構造的天井**: 現行モデルの上限は月商¥3〜4億（§3.5）。10億はモデル拡張前提＝経営判断が必要。
- **助成金依存**: 法人成約の8割超が助成金活用。制度変更・社労士(TRIPORT)連携が止まると成約率に直撃。
- **属人化**: 営業2名がボトルネック。AI一次対応とFC自走で分散する。
- **法務・景表法・特商法**: 教育/FC商材・助成金訴求は表記規制が厳しい。誇大表現を避け、人間が最終チェック。
- **AI出力の品質**: ハルシネーション対策として3モデル相互チェック＋人間承認を必須とする。
